Огляд проекту

Це інтернет-магазин розвиваючих іграшок та освітніх товарів для дітей, розроблений як статичний веб-сайт з інтеграцією сервісів для обробки замовлень.

Бізнес-логіка

1. Модель монетизації

  • B2C (Business to Consumer) - продаж товарів безпосередньо кінцевим споживачам
  • Без онлайн-оплати - замовлення обробляються вручну через форми
  • Lead Generation - збір контактів потенційних клієнтів через форму замовлення

2. Товарна політика

Каталог поділяється на категорії:

Розвиваючі конструктори
М'які іграшки
Освітні матеріали
Творчі набори

3. Система обробки замовлень

Використовується подвійна інтеграція:

EmailJS

  • Відправляє дані замовлення на email власника
  • Налаштовується через scripts/main.js
  • Потрібна реєстрація та створення шаблону листа

Web3Forms

  • Альтернативний метод відправки форм
  • Використовує Access Key
  • Інтегрується безпосередньо в HTML форму

4. SEO та маркетингова стратегія

  • Structured data (JSON-LD) для розширених сніпетів у Google
  • Meta теги для соціальних мереж (Open Graph, Twitter Cards)
  • Sitemap.xml та robots.txt для індексації
  • Оптимізовані зображення для швидкого завантаження

Технічна архітектура

Frontend структура:

scripts/
main.js → Основна логіка, обробка форм, EmailJS
catalog.js → Динамічне завантаження каталогу товарів
breadcrumbs.js → Навігаційні хлібні крихти
styles/
main.css → Глобальні стилі, layout
catalog.css → Стилізація каталогу та карток товарів
footer.css → Футер із контактами
order-section.css → Стилі форми замовлення

Ключові JavaScript компоненти:

main.js

  • Ініціалізація EmailJS
  • Валідація форми замовлення
  • Відправка даних на сервер
  • Обробка помилок

catalog.js

  • Рендеринг товарів із даних (можливо JSON/JS масив)
  • Фільтрація по категоріях
  • Динамічне створення карток товарів

breadcrumbs.js

  • Побудова навігаційного шляху
  • Відстеження поточної сторінки
  • Покращення UX
